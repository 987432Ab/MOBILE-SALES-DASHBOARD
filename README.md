# 📊 Mobile Sales Analytics Dashboard (MTD & YTD Performance Insights)

## 📌 Project Overview
This project presents an interactive **Mobile Sales Dashboard** built to analyze and visualize sales performance across multiple dimensions such as time, location, product categories, and payment methods.

The dashboard enables users to explore key business insights and make data-driven decisions effectively.

---

## 🎯 Project Objective
To design and develop an interactive dashboard that provides comprehensive insights into mobile sales performance across different dimensions such as time (Year, Quarter, Month), geography (City), product categories (Brand & Model), and payment methods.

The objective is to enable stakeholders to:
- Monitor Total Sales, Transactions, Quantity, and Average Price  
- Analyze Month-to-Date (MTD) and Year-to-Date (YTD) trends  
- Compare current performance with previous periods  
- Identify top-performing brands, models, and cities  
- Understand customer purchasing behavior and payment preferences  

This dashboard helps in data-driven decision-making, performance tracking, and identifying growth opportunities.
